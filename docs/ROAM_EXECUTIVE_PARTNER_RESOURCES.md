## 5. Roam 🧭 AI Sales Agent — Full Capabilities

### What It Is

Roam 🧭 is a **custom-built, wellness-trained AI sales agent** — not a plugin or off-the-shelf chatbot. It was purpose-built from the ground up as part of the Omni platform architecture. It operates 24/7 as a virtual wellness employee, handling the full customer journey from discovery to purchase.

### Core Capabilities

| Capability | Description |
|------------|-------------|
| **Lead Qualification** | Uses travel psychology to understand traveller type, destination, duration, and data needs |
| **Product Recommendation** | Matches customers to the optimal eSIM plan based on their specific journey profile |
| **Wellness-Literate Conversation** | Trained to understand retreat types, meditation practices, yoga styles, and holistic travel contexts |
| **Internal Routing** | Directs users exclusively to in-app store pages and checkout — never to external sites |
| **Multi-Currency Guidance** | Explains pricing in ZAR, USD, and EUR with real-time context |
| **Device Compatibility** | Guides users through eSIM compatibility checks for their specific phone model |
| **Discount Code Application** | Contextually offers and explains promotional codes based on customer journey |
| **Objection Handling** | Addresses common concerns (roaming costs, dual SIM, activation process) with empathy |
| **Post-Sale Support** | Assists with activation instructions, troubleshooting, and QR code delivery |
| **Communication Preference** | "Opening Ritual" that checks if the user prefers emojis or professional text formatting |

### Wellness Training Domains

Roam 🧭 is specifically trained across these wellness verticals:

- **Yoga & Meditation Retreats** — Understands Vinyasa, Hatha, Vipassana, silent retreats, etc.
- **Holistic Health & Healing** — Intuitive healing, Ayurveda, Traditional Chinese Medicine concepts
- **Plant-Based Nutrition** — Conscious food, wellness cuisine, retreat dietary requirements
- **Nature & Eco-Wellness** — Safari connectivity, nature immersion, eco-lodge requirements
- **Mental Health & Mindfulness** — Stress management, digital detox awareness, mindful connectivity
- **Financial Wellness** — Budget-conscious travel planning, value-per-GB calculations
- **Children's Wellness** — Family travel data requirements, parental control awareness
- **Animal Wellness** — Wildlife tourism, ethical safari practices
- **Community Impact** — Valley of Plenty project context, conscious consumerism

### Virtual Wellness Employee Framework

Roam 🧭 is part of a broader "Virtual Wellness Employee" system:

| Agent | Role | Deployment |

### What Makes It Custom-Built (Not a Plugin)

1. **Purpose-built edge function** — Runs on Supabase serverless infrastructure, not a third-party widget
2. **Custom prompt engineering** — Wellness-specific persona with travel psychology principles
3. **Powered by Google Gemini** — Uses advanced language model with structured response formatting
4. **Clean output formatting** — Processes responses to remove raw markdown artifacts for premium feel
5. **Integrated with store data** — Real-time access to product catalogue, pricing, and availability
6. **Automated notifications** — Triggers email and WhatsApp alerts on lead capture and sales
7. **Conversation persistence** — Full chat history stored in Supabase for analytics and training
8. **Commission tracking** — Each sale attributed with UTM parameters for accurate commission calculation

## 6. Technical Architecture Overview

### Platform Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Frontend | React + TypeScript + Tailwind CSS | Responsive, modern UI |
| Backend | Supabase Edge Functions (Deno) | Serverless API logic |
| Database | PostgreSQL (Supabase) | Structured data, RLS security |
| AI Engine | Google Gemini | Natural language sales agent |
| Payments | PayPal CardFields | Direct card payments (USD) |
| Email | Resend API | Transactional and marketing emails |
| Notifications | WhatsApp Business API | Real-time sale/lead alerts |
| Analytics | GA4 + Meta Pixel | Conversion tracking |
| eSIM Provider | RoamBuddy API | Product catalogue, order fulfilment |

### Integration Architecture

```
Customer Journey:
  Browser → AI Sales Bot (Roam 🧭) → Product Selection → Guest Checkout
    → PayPal Payment → RoamBuddy API (Order Creation) → QR/LPA Delivery
    → Email Confirmation → WhatsApp Notification → Commission Tracking
```

### Key Technical Differentiators

- **Guest checkout** — No login required; reduces friction for first-time travellers
- **Automatic token refresh** — Handles RoamBuddy API session management seamlessly
- **Multi-currency display** — ZAR, USD, EUR with real-time toggling
- **Instant fulfilment** — QR code and LPA string delivered immediately via email
- **Row-level security** — All database access governed by Supabase RLS policies
- **Edge-computed** — All server logic runs at the edge for global low-latency
